Tutorials and Learning

Tutorial 在 AgentGoRound 不是附屬說明文件，而是把產品能力變成可操作 journey 的核心 surface。YAML、runtime step evaluation、 UI checklist 與 real tutorial runner 共用同一份資料結構。

為什麼 Tutorials 很重要

  • 它讓「功能存在」變成「使用者真的能一步一步走完」。
  • 它讓 onboarding 與 regression test 共享同一套 scenario 定義。
  • 它把 agent、docs、skills、MCP、built-in tools 的組合關係具體化。

代表性案例

  • Case 5：使用 agent-browser MCP 讀取 GitHub Trending。
  • Case 6：使用多輪 skill 操作 GitHub Trending 並提取摘要。
  • Case 3：使用 built-in tools 做出可觀察的前端 UI。
