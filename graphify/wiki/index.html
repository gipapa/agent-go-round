AgentGoRound WIKI

這份 wiki 是基於 concept-first corpus 重做的 AgentGoRound 導覽版本。它刻意把焦點放在產品能力、runtime、 教學案例、MAGI、與前端呈現方式之間的關係，而不是單純列出程式檔案之間的 AST 依賴。

產品導覽頁 直接看線上導覽版介紹頁，快速理解 AgentGoRound 的定位、功能與使用方式。 PPTX 原始檔 查看或下載產品介紹投影片原始檔，適合做簡報、分享或延伸整理。 互動圖譜 用視覺化節點與社群關係快速看整個專案的概念連結。 純文字報告 看社群摘要、關鍵節點與 graphify 對這份專案的整體整理。 為什麼要有這份 Wiki 理解 graph / report / wiki 三種入口各自扮演的角色。

快速導覽

如果你是第一次看 AgentGoRound，建議先從產品定位開始，再一路往下看 routing、browser skill runtime、tutorial、MAGI， 最後再回來看 visualization / graphify。這條路徑最接近 README 原本的敘事方式。

Product Overview 先理解 browser-first、frontend-only 與 local-first 的產品定位。 Agent and Routing 理解 credential pool、load balancer 與 agent access control。 Browser and Skill Runtime 理解 multi-turn skill、browser observation 與 completion gate。 Tutorials and Learning 理解教學案例如何同時兼具 onboarding 與 test case。 MAGI and Special Modes 理解三賢人模式如何接管舊的 goal-driven 入口。 Visualization and Rendering 理解 dashboard、render anything、graphify 之間的關係。
